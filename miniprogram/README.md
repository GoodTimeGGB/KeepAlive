# 🚀 程序员加油站

这是一个为程序员设计的健康管理微信小程序，帮助你保持良好的工作习惯。

## 📋 项目简介

- **项目名称**: 程序员加油站
- **版本**: V1.0 演示版本
- **核心理念**: 护眼、舒适、极简、养生
- **数据存储**: 本地存储 (演示版本)

## ✨ 核心功能

### 1. 喝水提醒
- 记录每日饮水量
- 可视化进度显示
- 一键打卡，简单快捷

### 2. 运动提醒
- 提肛运动倒计时
- 久坐提醒
- 避免长时间久坐

### 3. 数据统计
- 连续打卡天数
- 今日进度
- 历史记录

### 4. 个性化设置
- 自定义提醒时间
- 工作时间设置
- 微信登录同步

## 🎨 设计风格

采用 **Brutalism (野兽派)** 设计风格，配合护眼配色方案：

- **主色调**: 橙色 `#FF6B35`
- **次色调**: 青色 `#4ECDC4`
- **背景色**: 深黑 `#0D0D0D`
- **特点**: 粗犷、直接、高对比度

## 📱 技术栈

- **前端框架**: 微信小程序原生开发
- **UI 方案**: 自定义 CSS (Brutalism 风格)
- **数据存储**: wx.setStorageSync (本地存储)
- **组件**: 自定义组件 (进度环、计时弹窗)

## 🚀 快速开始

### 1. 克隆项目

```bash
git clone <repository-url>
cd KeepAlive/miniprogram
```

### 2. 配置 AppID

打开 `project.config.json`，将 `appid` 修改为你的小程序 AppID。

```json
{
  "appid": "your_actual_appid_here"
}
```

### 3. 导入微信开发者工具

1. 打开微信开发者工具
2. 选择"导入项目"
3. 选择 `miniprogram` 目录
4. 填写 AppID
5. 点击"导入"

### 4. 运行项目

点击微信开发者工具的"编译"按钮，即可在模拟器中预览。

## 💡 使用说明

### 喝水打卡

1. 打开首页
2. 点击 "DRINK WATER" 按钮
3. 进度环会自动更新
4. 震动反馈确认

### 运动打卡

1. 打开首页
2. 点击 "MOVE / EXERCISE" 按钮
3. 弹出 60 秒倒计时弹窗
4. 倒计时结束后完成打卡

### 设置提醒

1. 进入设置页
2. 微信一键登录（可选）
3. 开启提醒开关
4. 设置工作时间段
5. 添加到日历（推荐）
